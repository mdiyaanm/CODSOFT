# CodSoft Data Science Internship

This repository contains the tasks completed during my Data Science Virtual Internship at CodSoft.

## Task 1: Titanic Survival Prediction
Built a machine learning model to predict whether a passenger survived the Titanic disaster using features like age, gender, and passenger class.

## Task 2: Iris Flower Classification
Used machine learning to classify iris flowers into three species based on sepal and petal measurements.

## Task 3: Sales Prediction
Developed a model to predict product sales based on advertising budget data.

## Tools Used
- Python
- Pandas
- NumPy
- Scikit-learn
- Matplotlib
- Jupyter Notebook
